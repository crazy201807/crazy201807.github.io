Hexo 基础功能

Hexo是一款免费的博客生成器，可以让你用简单的方式创建一个静态网站，包括博客文章、页面、标签和分类等内容。它是基于Node.js编写的，因此需要一定的编程知识，但它也提供了许多现成的主题和插件，可以帮助你快速搭建一个漂亮的博客。

使用Hexo可以将你的博客内容存储在Markdown文件中，通过Hexo的命令行工具可以将这些文件转换为HTML页面，并将它们发布到网站上。Hexo也支持多种语言和自定义域名，因此可以适应各种博客需求。

总之，如果你想快速搭建一个简单美观的博客，而且不想花费太多精力和时间，那么Hexo是一个非常不错的选择。

Hexo常见命令

  1. hexo new “post-title”：创建一篇新的博客文章，可以指定文章的标题。
  2. hexo generate：生成静态网页，包括 HTML、CSS、JS 等文件。
  3. hexo server：启动本地服务器，可以在浏览器中查看博客网站。
  4. hexo clean：清除之前生成的网页和缓存文件。
  5. hexo deploy：将博客网站部署到远程服务器或者 GitHub Pages 上。
  6. hexo publish “post-title”：发布一篇博客文章，可以指定文章的标题。
  7. hexo new page “page-name”：创建一个新的页面，可以指定页面的名称。
  8. hexo list：列出所有的博客文章和页面。
  9. hexo help：显示 Hexo 的帮助信息。

Next 主题配置

  1. 头像：您可以在配置文件中指定头像的 URL，以便在博客网站的侧边栏或文章页面中显示头像。
  2. 侧边栏：Next 主题提供了多种侧边栏风格和布局，您可以根据自己的喜好进行选择和定制。例如，您可以添加个人简介、社交媒体链接、最近的文章、标签云等内容。
  3. 文章页头：Next 主题支持在文章页头中显示作者信息、发布日期、阅读量等信息，您可以通过配置文件中的 post_meta 选项进行设置。
  4. 代码高亮：Next 主题集成了多种代码高亮方案，例如 Highlight.js 和 Prism，您可以选择喜欢的方案，并进行定制。
  5. 字体和配色方案：Next 主题支持多种字体和配色方案，您可以选择喜欢的方案，并进行定制。例如，您可以通过配置文件中的 font 和 scheme 选项进行设置。

插件

Hexo 的插件生态非常丰富，可以根据自己的需求选择安装不同的插件。以下是一些常用的、优秀的 Hexo 插件：

  1. hexo-deployer-git：将博客部署到 GitHub Pages 上的插件。
  2. hexo-generator-feed：生成 RSS 订阅的插件。
  3. hexo-generator-sitemap：生成网站地图的插件。
  4. hexo-renderer-markdown-it：使用 Markdown-it 引擎渲染 Markdown 的插件。
  5. hexo-wordcount：统计博客字数的插件。
  6. hexo-generator-json-content：生成 JSON 数据的插件，可用于构建搜索功能等。
  7. hexo-auto-category：自动根据文件夹路径生成文章分类的插件。
  8. hexo-generator-search：生成全文搜索的插件，可在博客网站中添加搜索功能。
